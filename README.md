# CarePlus

Projeto front-end estático do CarePlus — Agendamento Inteligente.

Conteúdo
- HTML/CSS/JS para interface de agendamento, login simples e dashboard.

Como rodar localmente
1. Instale Node.js (opcional) e `npx`.
2. No diretório do projeto, execute:

   npx http-server -p 3000

3. Abra http://127.0.0.1:3000

Login
- Login é feito via modal no /pages/auth/login.html e o nome é salvo em localStorage.

Deploy
- Este repositório pode ser publicado com GitHub Pages (branch `main`, diretório `/`).
